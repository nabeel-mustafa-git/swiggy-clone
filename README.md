<h3 align="center">Swiggy Clone</h3>

<p align="center">As a front-end developer passionate about crafting engaging user experiences, I've created this Swiggy-inspired platform (Actually a little Clone). Utilizing Swiggy's APIs, along with React hooks, custom hooks, React Router DOM, Context API, Redux, Redux Toolkit, and Tailwind CSS, I've designed a sleek and intuitive interface. While this is not a fully functional food delivery app, it showcases my skills in front-end development. Feel free to explore and enjoy the user-friendly design.</p>
